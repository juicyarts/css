# Grid

12 colum grid, currently no responsive options

## Examples

Default usage

```html
<div class="container">
  <div class="row">
    <div class="col-6 border-solid border-s border-color-gray-2 p-s">
      Col-6
    </div>
    <div class="col-6 border-solid border-s border-color-gray-2 p-s">
      Col-6
    </div>
  </div>
</div>
```

```html

  <div class="row">
    <div
      class="col-4 border-solid border-s border-color-gray-2 p-s text-align-center"
    >
      Col-4
    </div>
    <div
      class="col-4 col-offset-4 border-solid border-s border-color-gray-2 p-s text-align-center"
    >
      Col-4
    </div>
  </div>
</div>
```
